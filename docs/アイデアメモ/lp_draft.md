## 3. ソリューション (Solution)

### 🛡️ 「守るべき資産」を外に出さない
Ayato Transcriber は、OpenAI Whisper とローカル LLM (Gemma 等) をあなたの PC 内ですべて完結させます。
インターネットがオフラインの状態でも、録音から文字起こし、要約、チャット質問まで可能です。

### 🧠 蓄積された知見を「資産化」する
過去の全会議データを SQLite FTS5 高速検索エンジンでインデックス化。
RAG (検索拡張生成) テクノロジーにより、AI があなたの代わりに過去の議事録から答えを探し出します。

### 🚀 ゼロ・コンフィギュレーション
複雑なセットアップは不要です。exe を実行するだけで、AI モデルのダウンロードから環境構築まで自動で行われます。

## 4. 特徴的な機能 (Key Features)

1.  **リアルタイム & ファイル文字起こし**: システム音声やマイク入力をその場でテキスト化。
2.  **ビジュアル・コンテキスト抽出**: 動画内のスライド変化を検知。音声だけではわからない「視覚情報」を AI が理解。
3.  **マルチプロバイダー対応**: ローカル推論 (Gemma) と、高品質な Google Gemini API を自由自在に使い分け。
4.  **ハードウェア最適化**: あなたの PC の RAM/VRAM を検知し、最適なモデル (1B/4B/8B/70B) をレコメンド。
